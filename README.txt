== Description ==

Calculates an average required time to complete reading a post.

== Installation ==

1. Upload the entire estimated-post-reading-time folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. In the admin panel, go to Settings -> "Post Reading Time" and change the "Words Per Minute" option to what you want (default is 250).
4. Use shortcode [est_time] in any post/page content to display the estimated post reading time.

== Frequently Asked Questions ==

= Which is the keyword of the shortcode? =

The keyword of the shortcode is [est_time]
