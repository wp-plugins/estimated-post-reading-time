== Description ==

Calculates an average required time to complete reading a post.

== Installation ==

1. Upload the entire estimated-post-reading-time folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Under the "Estimated Time" option of the admin menu, change the option in the way you want

== Frequently Asked Questions ==

= Which is the keyword of the shortcode? =

The keyword of the shortcode is [est_time]
